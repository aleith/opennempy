Getting Started¶

Requirements¶

  • Python 3.8+ (see .python-version with pyenv)

Quickstart¶

In an existing project:

$ pip install opennem

With poetry:

$ poetry add opennem

Usage¶

>>> import opennem
>>> opennem.api.networks()
